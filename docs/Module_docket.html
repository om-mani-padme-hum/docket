Parser.onComment(block, text, start, end)

Added in: v0.1.0
Updated in: v0.2.0

  • block - <boolean>
  • text - <string>
  • start - <number>
  • end - <number>

This is a handler that processes comments found by the acorn JavaScript parser. It is used to identify and parse docket entries and stores the resulting object tree in the parser for later document generation.

Parser.parseFile(filePath)

Added in: v0.1.0
Updated in: v0.2.0

  • filePath - <string>

Parses all docket entries in the modules, classes, and method and/or function signatures of the file located at filePath and stores the resulting object tree in the parser for later document generation.

Parser.parseFiles(fileList)

Added in: v0.2.0

  • fileList - <Array>

Parses all docket entries in the modules, classes, and method and/or function signatures of the files in the fileList array and stores the resulting object tree in the parser for later document generation.
